# Sound Assets

Place your notification sound file here.

**Required file:**
- `ride_request_notification.mp3` or `ride_request_notification.wav`

**Requirements:**
- Format: MP3 (recommended) or WAV
- Duration: 1-3 seconds
- Volume: Moderate (not too loud/annoying)
- Sample rate: 44.1kHz recommended

**Example sources:**
- Free notification sounds: freesound.org, zapsplat.com
- Or create your own notification sound
